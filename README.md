This plugin prevents a login packet crash by users sending a massive chain payload. This also prevents the login packet from being spammed and includes anti-bot protection.

Other features:
  Prevents BDS from sending player tp and block_event locations allowing people to locate bases
  Anti-horion crasher
  Anti-device spoofer
  Anti-HWID spoofer
  Discord Alerts


Config:
```json
{
    webhook: "",
    icon: ""
}
```

Goes inside plugins/data/config.json
